Benry-CmdOpt

Benry-CmdOpt is a command option parser for Ruby. Much better than optparse.rb (Ruby standard library).

Why not optparse.rb? In short, because it has several design mistakes. Read this document for details.

Benry-CmdApp

Benry-CmdApp is a Ruby framework for command-line application that takes subcommands.

If you want to create a command application like git, docker, or npm, Benry-CmdApp is the solution.
